#Xitu
本项目仅仅用户学习目的,在Android Studio下使用`单元测试`,以及使用`开源库`进行快速开发
## 单元测试
* junit4
使用到最多,配合`retrofit`和`rajava`进行restapi请求测试
* mockito
* hamcrest
